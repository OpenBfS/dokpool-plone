docpool.theme Installation
--------------------------

To install docpool.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``docpool.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        docpool.theme
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
